# Open source license

ReviewInbox uses Apache-2.0 for the repository. This favors low-friction adoption by indie developers, small teams, and companies over stronger SaaS-clone protection from copyleft licenses such as AGPL.

## Consequences

The open source product should stay genuinely useful without proprietary folders or artificial limitations. The hosted cloud service will compete through convenience, operations, onboarding, support, and managed delivery rather than license restrictions.
